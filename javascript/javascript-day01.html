selector

  • getElementsById

    ID 값을 가져오는데 사용

  • getElementsByClassName

    NAME 값을 가져오는데 사용

  • getElementsByTagName

    태그(Tag)값을 가져오는데 사용

  • querySelectorAll

    타겟을 배열형식으로 모두 받아옴

  • querySelector

    오직 첫번째 요소만

  • parentElement

    부모 요소에 접근

  • parentNode

    부모 노드에 접근, 비 요소 노드를 포함

  • children

    자식 요소에 접근

  • childNodes

    자식 노드에 접근, 비 요소 노드를 포함

  • firstChild

    공백도 자식.. first childe node를 element node 또는 text node, comment node로 반환

  • firstElementChild

    first child node를 element node로만 반환을 한다.(text와 comment nodes를 무시)
    textContent에 접근하거나 style을 바꾼다든지 등등 HTML code에 반영을 위한 접근이 필요하다면 firstElementChild을 쓴다.

  • lastChild

  • lastElementChild

  • previousSibling

  • previousElementSibling

    같은 깊이에 있는 자매 노드 중 본인 노드 기준으로 이전 자매 노드에 대한 접근

  • nextSibling

    본인 노드 기준으로 다음 자매 노드에 대한 접근

  • nextElementSibling

    본인 노드 기준으로 다음 자매 노드에 대한 접근, HTML code에 반영을 위한 접근이 필요할 때에는 nextElementChild를 쓴다.

  • creatElement

    지정한 tagName의 HTML 요소를 만들어 반환

  • nth-child(n)

    부모 엘리먼트의 모든 자식 엘리먼트중 n번째

  • nth-of-type(n)

    부모 엘리먼트의 특정 자식 엘리먼트중 n번째

  • first

    결과셋을 기준으로 한 위치

  • first-child

    부모와 형제간의 관계에 기반한 위치
